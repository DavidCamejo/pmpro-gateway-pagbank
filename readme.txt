== Descripción ==
Integración de PagBank como gateway de pago para Paid Memberships Pro. Este plugin permite procesar pagos con tarjeta de crédito, Pix y Boleto bancario a través de la plataforma PagBank (anteriormente PagSeguro).

== Características ==
* Procesa pagos únicos y recurrentes
* Soporte para tarjeta de crédito, Pix y Boleto
* Webhooks automáticos para actualización de estado de pagos
* Compatible con Paid Memberships Pro

== Instalación ==
1. Sube la carpeta `pmpro-gateway-pagbank` a `/wp-content/plugins/`.
2. Activa el plugin en WordPress.
3. Configura las credenciales de PagBank en "Memberships > Configuración de Pagos".
